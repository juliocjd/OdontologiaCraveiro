# Odonto em Casa

Site institucional em React/Vite para atendimento odontológico domiciliar e
hospitalar em Maringá e região.

## Como rodar

```bash
npm install
npm run dev
```

## Rotas públicas

- `/`
- `/servicos/`
- `/atendimento-domiciliar/`
- `/atendimento-hospitalar/`
- `/idosos-e-acamados/`
- `/atendimento-infantil/`
- `/cidades-atendidas/`
- `/instituicoes-e-cuidadores/`
- `/escolas/`
- `/sobre/`
- `/faq/`

## Build

```bash
npm run build
```

Output directory:

```bash
dist
```

Conteúdo, metadados e mensagens de WhatsApp ficam em `src/data/siteData.js`.
